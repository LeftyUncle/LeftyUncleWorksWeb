cWSクラス プロパティ

プロパティ データ型 説明
TargetBook WorkBook cWSクラスで操作を行いたいWorkBookオブジェットを指定します。
既に値が設定されているときは、そのWorkBookオブジェクトを返します。
このプロパティは一度指定すると、次に指定するかメソッドの引数で指定しない限り値は変わりません。
TargetSheet WorkSheet cWSクラスで操作を行いたいWorkSheetオブジェットを指定します。
既に値が設定されているときは、そのWorkSheetオブジェクトを返します。
このプロパティは一度指定すると、次に指定するかメソッドの引数で指定しない限り値は変わりません。
TargetAddr String cWSクラスで操作を行いたいWorkSheetのセルのアドレスをA1形式で指定します。
既に値が設定されているときは、その値をを返します。
このプロパティも次に指定するまで値を保持しますが、TargetBookやTargetSheetが変更されると値の意味は失われます。
LastErrMsg String cWSクラスで最後に発生したエラーメッセージを返します。
ValueErrorReturn String セル値を返すメソッドでエラーが発生した時に返される値です。
AddrErrorReturn String A1形式のアドレスを返すメソッドでエラーが発生した時に返される値です。
StringErrorReturn String 文字列を返すメソッドでエラーが発生した時に返される値です。
NumErrorReturn Long 数値を返すメソッドでエラーが発生した時に返される値です。
ActiveBook WorkBook 現在アクティブになっているWorkBookオブジェクトを返します。
ActiceSheet SorkSheet 現在アクティブになっているWorkSheetオブジェクトを返します。
ActiveAddr Atring 現在アクティブになっているWorkBook.WorkSheetのA1形式のアドレスを返します。
SaveBook WorkBook CurStateSaveメソッドで保存したWorkBookオブジェクトを返します。
SaveSheet SorkSheet CurStateSaveメソッドで保存したWorkSheetオブジェクトを返します。
SaveAddr Atring CurStateSaveメソッドで保存したA1形式のアドレスを返します。
ThisExcelVersion Long このExcelのバージョンを数値で返します。
バージョン 値
Excel5.0 5
Excel95 95
Excel97 97
Excel2000 2000
Excel2002 2002
Excel2003 2003
Excel2007 2007
Excel2010 2010
FormatTypeString String セルの表示形式を「文字列」にします。SetFormatTypeメソッドで使用します。
FormatTypeNumeric Sring セルの表示形式を「数値」にします。SetFormatTypeメソッドで使用します。
FormatTypeNomal String セルの表示形式を「標準」にします。SetFormatTypeメソッドで使用します。
Excel2003までは256列、Excel2007〜2010は16,384列です。
MaxColNum Long このExcelのバージョンの最大列数を整数で返します。
Excel2003までは256列、Excel2007〜2010は16,384列です。
MaxLine Long このExcelのバージョンの最大行数を整数で返します。
Excel2003までは65,536行、Excel2007〜2010は1,048,576行です。
