2012-11-13 Windows Media Audio でストリーミング配信のウェブページを作ったららこうなった

「ストリーミング再生」は基本的に「動画」が対象で、音声のみのストリーミングは「Real」ですかね。
ストリーミングと言えば、もっぱら「flv」か「mp4」ですね。
「Real」なんて最近滅多に見かけません、プレイヤー等々、最近はインストールしていない人も多いのでは？
私もその一人です。
レンタルしている「さくらインターネット」は容量が「１GB」しかないため「ストリーミング動画」をアップするいには絶対的に容量が小さすぎて無理なんです。
で、ちょいと調べてみるとWindows Media Audioでもストリーミング出来るんですね。
ストリーミング用のファイル変換も「Windows Media エンコーダ x64 Edition」で簡単にできます。
ＩＥ１０で実験してみたらすんなり成功、リンクをクリックしたらすぐにWindows Media Playerが起動して再生が始まります。
ですが、FireFox、Chermeではストリーミング再生してくれません、ダウンロードしに行っちゃいます。
確認したバージョンは、FireFox「16.0.2」、Cherme「バージョン 23.0.1271.64 m」
Windows8ではプラグインをインストールしたら両方とも解決しました。
尚、Cherme用のプラグインはまだ準備されていないようです、ダウンロードできません。

＜訂正＞
Cherme用のWindows Media Player Pluginはリリースされています。
詳しくは「備忘録 Windows Media Player Plugin ダウンロードとインストール」をご覧ください。
＜/訂正＞

この２つのブラウザのストリーミング再生、ちょっと便利悪い。
プレイヤーが起動するんじゃなくて、新しいタブが生成されて波形動画が表示されます。


同じバージョンのブラウザでも、WindowsXP(Pro SP3)では、異なる動きをします。
Windows Media Playerは「１１」をダウンロードしてインストールしました。
FireFox用のプラグインもインストール済です。
WindowsXPの場合はWindow Media Player11が起動されます。


伊豆の音楽師匠の環境（XP＋FireFox）ではストリーミング再生がなかなかうまく行かないみたいで手こずってます。
でもVMWareでWindowsXPインストールしたらMedia Player 9 で何もせんでもストリーミング再生するし、相変わらずWindowsは困ったもののです。
